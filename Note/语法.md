# 语法

## 字面量

- **数字（Number）** 字面量 可以是整数或者是小数，或者是科学计数(e)。
  3.14159   123e5
- **字符串（String）** 字面量 可以使用单引号或双引号
  "hello"  'world'
- **表达式** 字面量用于计算
  1 + 2 * 3
- **数组（Array）** 字面量 定义一个数组
  [1, 2, 3]
- **对象（Object）** 字面量 定义一个对象
  {name: "John", age: 30, city: "New York"}
- **函数（Function）** 字面量 定义一个函数
  function myFunction(a, b) { return a * b;}

## 变量

关键字 `var` 来定义变量， 使用`=`来为变量赋值

```javascript
var x,length;
x=5;
length=6;
```

- **变量通常是可变的，字面量是一个恒定的值**
